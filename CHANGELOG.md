# Changelog

## 0.2.0

- Community-ready release baseline mit GHCR-Publishing, GitHub Actions CI und oeffentlichem Unraid-Template auf Registry-Basis.
- Community-Health-Dateien fuer Bug Reports, Feature Requests, Pull Requests, Code Owners und Verhaltensregeln.
- Dockerfile mit OCI-Metadaten und veroeffentlichungsfaehigem Image-Layout.
- Dokumentierter Community-Endzustand und klarere Release-/Installationspfade fuer externe Nutzer.
- Fairere Paperless-KIplus-Validierungsregeln, damit konservative und fachlich plausible Antworten nicht unnoetig als Benchmark-Fehler enden.

## 0.1.0

- Initiale Version des Docker- und Unraid-tauglichen LLM-Benchmark-Orchestrators.
- Unterstützung für mehrere OpenAI-kompatible Modellendpunkte.
- Beispiel-Suites für Core, API-Replacement und Long-Context.
- Rohdatenexporte sowie Markdown-, HTML- und JSON-Reports.
- Unit-Tests für Loader, Validierung, Scoring und Reporting.
